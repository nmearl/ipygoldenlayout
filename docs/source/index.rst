
ipygoldenlayout
=====================================

Version: |release|

IPyWidget wrapper for the Golden Layout JavaScript package


Quickstart
----------

To get started with ipygoldenlayout, install with pip::

    pip install ipygoldenlayout

or with conda::

    conda install ipygoldenlayout
